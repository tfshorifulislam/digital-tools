#  Digital Tools Platform

## Project Description

Digital Tools Platform is a modern web application where users can explore and purchase various digital products such as templates, tools, and resources. It provides a smooth user experience with dynamic product display, cart functionality, and responsive design.

------------------------------

## Technologies Used

* React.js
* Tailwind CSS
* DaisyUI
* JavaScript (ES6+)
* React-Toastify (NPM Package)
* JSON (for product data)

------------------------------

## Features

* 🛒 Add to Cart system with real-time updates
* 🔄 Toggle between Products and Cart view
* 🔔 Toast notifications for user actions (e.g., add to cart)
